GIT COMMANDS

  • git add (file)
    • Stages file for commit
  • git status
    • Prints current status
  • git checkout -b (name)
    • Creates and goes to new branch
  • git branch -d (name)
    • Deletes branch
  • git checkout (name)
    • Moves to branch
  • git pull origin
    • Pulls files from origin
  • git push origin (branch)
    • Pushes branch to origin
  • git push --all origin
    • Pushes all branches to origin
  • git commit -m "message"
    • Commits with message
  • git init
    • Creates new local repository
  • git clone (URL)
    • Clones repository from source
  • git remote add (name) (URL)
    • Adds Remote
  • git branch
    • Lists all branches in the repository
  • git checkout -- (file_name)
    • discard changes in working directory
  • git grep "search"
    • Searches for "search"
  • git reset HEAD (file)
    • removes file for commit
  • git log
    • shows log of commits

COMMAND LINE COMMANDS

  • cd
    • Change directory
  • ls
    • List
  • cd ..
    • Go up one directory
  • pwd
    • Print working Directory
  • rmdir
    • remove directory
  • pushd
    • Push Directory
  • popd
    • Pop Directory
  • cp
    • Copy a file or directory
  • mv
    • Move file or directory
  • less
    • Page through a file
  • cat
    • Print entire file
  • find
    • Find Files
  • grep
    • Find inside of files
  • env
    • Look at environment
  • touch
    • Creates File
  • echo
    • prints some arguments
  • exit
    • Exits the shell
